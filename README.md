# Tracker

Bootstrapped:
- [x] Landing
- [x] Signed in
- [ ] Form for new sign in
- [ ] List of trackers
- [ ] Positions of tracker (with 'ping' button)

### Requirements
Runs on Python 3
Install the following with `pip3`:
```
pymysql
twilio
flask
runenv
```
### Usage
Fill in `.env_sample`, rename it to `.env`

### Run
```
python3 app.py
```
Point to `localhost:8000`
